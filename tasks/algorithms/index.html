Algorithms Preparation

Here you'll find a collection of algorithm problems to sharpen your problem-solving skills.

Algorithm Tasks

  • Two Sum Problem

    Given an array of integers, find two numbers that add up to a specific target. This is a fundamental problem to practice using hash maps and solving in O(n) time.

  • Reverse String

    Write a function to reverse a string. This problem helps with understanding pointers and string manipulation in-place.

  • Merge Sorted Arrays

    Merge two sorted arrays into one sorted array. This task is great for practicing sorting algorithms and understanding time complexity.

  • Fibonacci Sequence

    Implement an algorithm to generate the Fibonacci sequence up to a certain number. This classic problem is useful for understanding recursion and dynamic programming.
